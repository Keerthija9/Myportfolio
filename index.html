Gasoline Pricing Analysis

I accomplished a data mining project, analyzing fuel price variations using the ETL process with data sets from sources such as GasBuddy and the EIA (436 records, 12 variables), extracting, cleaning, and transforming data with Pandas by addressing missing values, encoding categorical variables, and engineering features like tax-adjusted prices. Exploratory data analysis (EDA) was performed using Matplotlib and Seaborn, with data visualizations such as scatter plots, histograms, and correlation matrices used to identify trends. Applied machine learning techniques to data models, including Forward Selection Linear Regression, PCA-enhanced Linear Regression, and Random Forest Regression (achieving the best accuracy with Random Forest based on RMSE and MAE), and performed K-means clustering to segment the dataset into four distinct clusters, uncovering market-specific pricing behaviors influenced by economic and geographic factors.
